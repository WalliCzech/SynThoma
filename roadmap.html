Kyberpunkový plán

✅ Hotovo / Stabilní

🎛️ Základní architektura

FastAPI backend s endpointy pro generování, lipsync, upscale a analýzu. Desktop GUI (Py + CustomTkinter). Statická neonová HTML stránka. Konfigurovatelný config.yaml, podpora presetů, logování a automatická detekce GPU/CUDA. 🛠️ - Ano, všechno je propojené. Ano, funguje to. Ne, nechci to vysvětlovat znova.

🧠 Lokální inference modelů

Plně funkční modely: Flux.1Dev, SDXL (UNet + VAE), WAN 2.1 i2v, GFPGAN, RealESRGAN, CodeFormer, Wav2Lip, RetinaFace, YOLOv5, S3FD, Dlib landmarks, GroundingDINO, SAM, ParseNet. Pixely pod kontrolou! 🌌 - Ne, nepotřebuješ internet. Jen pořádný železo. A klid v duši.

🧠 FluxPipeline – Vlastní generátor

Vlastnoručně vytvořený inference engine bez Diffusers: ručně stavěný Flux UNet (transformerový), CLIP/T5 encodery, VAE dekodér a DDIM scheduler. Optimalizace pro bfloat16, vlastní guidance, checkpointování a extrémní modularita. Slouží jako základ pro FluxRunner. 🚀 - Nezkompiloval jsem si to z GitHubu. Napsal jsem si to sám. A funguje to líp.

🧪 Specialitky

Facial expression editor s warp netem, segmentace pozadí/objektů, lipsync z audia/textu, forenzní moduly (deeDet + metadata), FaceSwap (InSwapper ONNX), 4x AI upscaling s real-time náhledem. 😎 - Když to pustíš, tvoje GPU začne sténat slastí.

🧪 Právě ve vývoji

🖼️ GUI / UX

Nový intuitivní layout GUI, drag & drop pro média, automatická synchronizace presetů, split view preview (původní vs. výstup). Ovládání hladší než kyberkatana! 🎮 - Už to nebude vypadat jako tabulka v Excelu. Bude to hezký. Fakt.

🧠 Backend & optimalizace

Docker image pro cloud API, oddělené task runnery, refaktor core komponent, asynchronní fronta požadavků. Backend rychlejší než neonový blesk! ⚡ - Až to spadne, nebude to moje chyba. Bude to elegantní pád.

📤 Webové rozhraní

Minimalistické UI s drag & drop, limity bez registrace (3 výstupy/den), BuyMeACoffee API pro donor funkce, realtime generování, sdílení přes URL. 🌐 - Nechceš platit? Tři výstupy a čus. Chceš víc? Kup mi kafe.

📱 Mobilní výhled

MVP mobilní appky (Flutter/React Native): upload, lipsync, preview, notifikace. HTTP API komunikace, příprava .apk pro Android. Kyberpunk v kapse! 📲 - Jo, bude appka. Jo, poběží na Androidu. iOS? Nejspíš ne, nechce se mi.

🕹️ Fun features

„Kyberpunkové výrazy“ – náhodné neonové presety. Automatický výběr snímků, „Cybersoul“ režim (video + lipsync + glitch), OBS plugin pro live výrazy. 😈 - Pro ty, co se nudí rychle. Neonové výrazy místo osobnosti.

🔮 Plány do budoucna / Vize

☁️ Cloudová inference

Napojení na RunPod (GPU inference přes REST), správa API klíčů, hybridní režim (lokální + cloud), automatická fronta při přetížení. AI bez limitů! 🌩️ - Chceš to pustit i na šunce? Pošli to do cloudu. Tam se to vypaří.

🧩 Modulární systém

Pluginová architektura pro třetí strany, profilování výkonu, GUI builder pro pipeline editor. Tvoř si vlastní kyberpunkový workflow! 🛠️ - Když ti to nebude stačit, naprogramuj si vlastní. Ale hezky bokem.

📚 AI školení / tutor

Režim „uč se s AI“ – vizuální kroky generace, interaktivní popisy, podcast/video série o architektuře. Uč se hackovat realitu! 🎓 - Konečně ti AI vysvětlí, co vlastně dělá. A nebude přitom zívat.

🌐 Komunita

Veřejný Discord, galerie uživatelských výstupů, soutěže o kybervidea, badge systém pro přispěvatele. Společně rozsvítíme kyberprostor! 🤝 - Nejsi sám. Je nás víc. A máme neonové výrazy. Přidej se, nebo zůstaň v šedi.
